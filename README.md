# Bone Weight Modifier
A non-destructive tool to add and edit bone weights for VRChat avatars.

You can assign new bone weights or overwrite existing ones.  
Since this is a non-destructive tool, you can always revert changes simply by removing the component.

![Bone Weight Modifier](https://github.com/user-attachments/assets/5ba67504-b7e0-4c10-a5f3-b38d7bb8b9b4)

## Installation
1. Press `Add to VCC` button in [VPM Listing](https://vpm.nekobako.net) and add the repository.
2. Press `Manage Project` button at your project.
3. Press `+` button at the right of `Bone Weight Modifier` package.

## Usage
1. Select an existing bone, or create an empty Game Object in your avatar.
2. Add the `Bone Weight Modifier` component.
3. Set the target bone and renderer.  
If the bone left unset, the target defaults to the object which the component is attached to.
4. Add and configure bone weight definitions.

For more details, refer to the [documentation](https://bone-weight-modifier.nekobako.net).
