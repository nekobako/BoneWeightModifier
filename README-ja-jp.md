# ボーンウェイトモディファイヤー
VRChat アバターのための非破壊ボーンウェイト追加/編集ツールです。

新規のボーンウェイトの割り当てや既存のボーンウェイトの上書きが行えます。  
非破壊ツールなのでコンポーネントを削除するだけでいつでも元に戻すことができます。

## インストール
1. [VPM Listing](https://vpm.nekobako.net) の `Add to VCC` ボタンを押してリポジトリを追加します。
2. プロジェクトの `Manage Project` ボタンを押します。
3. `Bone Weight Modifier` パッケージの右にある `+` ボタンを押します。

## 使い方
1. 既存のボーンを選択するか、アバターの中に空の Game Object を作成します。
2. `Bone Weight Modifier` コンポーネントを追加します。
3. 対象のボーンとレンダラーを設定します。  
ボーンを設定しない場合、コンポーネントがアタッチされているオブジェクトが対象となります。
4. ボーンウェイトの定義を追加、設定します。

詳しくは [ドキュメント](https://bone-weight-modifier.nekobako.net) を参照してください。
