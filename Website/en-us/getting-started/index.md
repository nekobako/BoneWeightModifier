# Getting Started
Bone Weight Modifier is a non-destructive tool to add and edit bone weights for VRChat avatars.

<video muted autoplay loop playsinline src="../videos/getting-started/bone-weight-modifier.mp4"></video>

You can assign new bone weights or overwrite existing ones.  
Since this is a non-destructive tool, you can always revert changes simply by removing the component.
