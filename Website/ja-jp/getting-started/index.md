# はじめに
ボーンウェイトモディファイヤーは、VRChat アバターのための非破壊ボーンウェイト追加/編集ツールです。

<video muted autoplay loop playsinline src="../videos/getting-started/bone-weight-modifier.mp4"></video>

新規のボーンウェイトの割り当てや既存のボーンウェイトの上書きが行えます。  
非破壊ツールなのでコンポーネントを削除するだけでいつでも元に戻すことができます。

まずはツールのインストールについて説明します。
